# Rock, Paper, Scissors (C#)

A simple Rock, Paper, Scissors game made in C# using a console interface.

### Features

- Player vs Computer gameplay
- Random choice generation for the computer
- Input validation
- Win / Lose / Draw logic
- Replay system

# How it works

### The game flow:

1. Player chooses:
   - Rock
   - Paper
   - Scissors
2. The computer randomly selects an option
3. The game compares both choices
4. Displays the result:
   - Win
   - Lose
   - Draw
5. Player can choose to play again or exit

# Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
